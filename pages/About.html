About The project

Many universities and colleges worldwide suspended classroom teaching due to the novel coronavirus pandemic, students opted for Computer science are facing lot of difficulties for practical subjects. Since there is no access to college computer lab, it is difficult for them to do coding or learn coding especially in India. One has to install Compilers or IDE in their system to do these. For those who does not have a PC or Laptop there is no way for doing it. They just have to rely only on the lessons taught virtually. So, we develop software as a service, a Cloud based lite weight web app for the students to do there coding works and get outputs. This app's main goal is that they can easily write programs, compile and debug them online.
This project is proposed for the betterment of student career. This project aims to help those students not have a PC or Laptop to do there coding works and get outputs. This research main goal is that they can easily write programs, compile and debug in online. Using these cloud compilers, we can reduce the storage space and movability, centralized compiler using cloud computing that helps to save the time, storage and cost using this concept. So that it’s a most appropriate tool to compile the code, remove the errors and debug it. The online IDEs available are heavy (Contains lot of scripts, Ads, Compatibility issues with mobile etc.) and most of the time they won’t even load on mobile browsers. If the student does not have access to high-speed internet then thy can’t even use the online IDEs. Also, the tracking, logging, advertisements, scripts everything is big privacy threat. We endorse privacy. So, the Binary squirrel will be open source and it will never have any ads, tracking, or cookies that can identify or target users.
